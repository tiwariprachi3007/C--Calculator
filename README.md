# Calculator (C)

## Overview
A simple C program that performs basic arithmetic operations (addition, subtraction, multiplication, division).  
This project demonstrates user input handling, switch-case logic, and modular programming in C.

## Features
- Addition, subtraction, multiplication, division
- Input validation for division by zero
- Menu-driven interface
- Clean and beginner-friendly code

## Usage
Compile and run:

```bash
gcc calculator.c -o calculator
./calculator
